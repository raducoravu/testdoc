# The ZIGI Local Repositories Panel

After the preparatory work above, the main ZIGI ISPF panel is presented, which lists all of the local Git repositories that have been added to ZIGI:

![](media/img(9).png)

There are both commands and row selections available for use. The commands are available via the Action Bar menu and the row selections are available by entering a / in the row selection. This means you can move the cursor to them and press the Enter key or move the cursor there using your mouse and double-click with the left mouse button \(assuming your TN3270 emulator supports that feature â€“ and most do but you may have to enable that feature\).
